# LeetCode Resources

## Repository Structure

- `problems/{number}-{slug}/solution.py` - LeetCode solutions (Python)
- `problems/{number}-{slug}/README.md` - Problem notes and approach
- `problems/misc/` - Non-LeetCode utility code
- `patterns/` - DSA pattern implementations in 7 languages
- `leetcode-journal/` - Algorithm-specific learning logs
- `images/` - Repository assets
